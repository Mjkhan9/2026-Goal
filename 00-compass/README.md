# 00 — Compass

Evergreen source of truth. These documents change **rarely** and define the *why* and the *how-at-a-high-level* of the entire initiative.

## Files

- `north-star.md` — mission, thesis, 12-month intent.
- `operating-principles.md` — non-negotiable rules of engagement.
- `roadmap.md` — phased arc: service → system → recurring → product.
- `glossary.md` — shared vocabulary so humans and AI use the same words.

## Rules for this folder

- Update only when the underlying **thesis or principle** changes.
- Keep each file short, declarative, and readable in under two minutes.
- Do not put weekly updates, research notes, or execution details here — those belong in `01-state/` or the relevant domain folder.
