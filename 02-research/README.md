# 02 — Research

Market and business intelligence. The purpose of this folder is to turn
curiosity into conviction — enough real signal to justify an offer direction.

## Primary working surface (Phase 1)

`houston-smb-diagnostics/` — per-business diagnostic briefs on Houston-area
SMBs. This is where Phase 1 work lives. See its README for workflow.

## Other subfolders

- `niches/` — one file per niche under evaluation. Include size, pain,
  buying power, accessibility, fit. Populated once diagnostics surface a
  clear pattern worth deeper analysis.
- `markets/` — broader market context when it spans multiple niches.
- `competitors/` — one file per competitor or reference business.
- `signals/` — trends, data points, quotes, discovery conversation logs,
  observations. Cite sources. Dated.

## Rules

- One topic per file. Filename in `kebab-case`.
- Always cite sources. Undated/unsourced research is noise.
- Summarize conclusions at the **top** of each file.
- Promote validated opportunities to `03-offers/opportunities/` only after
  diagnostic patterns and at least one discovery conversation support it.
