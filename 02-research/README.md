# 02 — Research

Market and niche intelligence. The purpose of this folder is to turn **curiosity into conviction** — enough signal to commit to an offer direction.

## Subfolders

- `niches/` — one file per niche under evaluation. Include size, pain, buying power, accessibility, fit.
- `markets/` — broader market context when it spans multiple niches.
- `competitors/` — one file per competitor or reference business. What they do, pricing, positioning, weaknesses.
- `signals/` — trends, data points, quotes, screenshots, observations. Cite sources.

## Rules

- One topic per file. Filename in `kebab-case`.
- Always cite sources. Undated/unsourced research is noise.
- Summarize conclusions at the **top** of each file, not buried at the bottom.
- Promote validated opportunities into `03-offers/opportunities/`.
