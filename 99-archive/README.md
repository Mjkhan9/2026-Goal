# 99 — Archive

Parked or historical material. Nothing here is active — but nothing here is deleted either.

## Rules

- Move items here when they are no longer active but may have future reference value (retired offers, old research, superseded playbooks, completed client engagements after a long dormancy).
- Preserve the original folder structure inside the archive where useful (e.g. `99-archive/offers/…`, `99-archive/research/…`).
- When archiving, add a short note at the top of the file explaining **why** it was archived and when.
- Do not prune unless the material is truly worthless. Space is cheap; lost context is expensive.
